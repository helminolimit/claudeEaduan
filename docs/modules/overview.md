# e-Aduan System — Project Overview

e-Aduan is a public complaints management system that allows citizens to submit complaints (aduan) to government agencies. Officers review and manage these complaints, while administrators oversee the entire system.

---

## User Roles

### Complainant (Pengadu)
- Register and login
- Submit aduan
- Track complaint status
- View complaint history
- Upload evidence

### Officer (Pegawai)
- View assigned complaints
- Update complaint status
- Add comments
- Assign priority

### Admin (Pentadbir)
- Manage users
- Manage officers
- View all aduan
- Generate reports
- System settings
